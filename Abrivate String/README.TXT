Summary
=======
    Create an abbreviated version of a string given a dictionary.

Constraints
===========
    - Code must be written in C.
    - No third party libraries may be used.
    - Use only the C standard library.

Files Provided
==============
    abbr.h: header file
    abbr.c: dummy implementation file
    main.c: test driver
    CMakeLists.txt: sample CMake file (you do not need to use this)

File Expected
=============
    abbr.c

Details
=======
   - Implement the function 'abbreviate_string' in abbr.c.  See abbr.h for function declaration and description.
   - For testing we will replace main.c with a fully functional test driver and will only be using your abbr.c/.h files
